採用した設計思想

地図が主役、解説は物語として沿わせる構成: 上半分に地図、下半分を「土地のたより」カードにしました。画面を開いた瞬間に「いま自分がどこにいて、どう動いてきたか」が視覚的に入り、続いて読み物として解説が目に入る流れです。
ダークテーマを旅の深みに活用: #0f0f10 という計画書の #1a1a1a より一段深い黒をベースにし、テキストは純白ではなく #f5f5f7 に落として、夜行列車の車内でも目に優しい明度にしました。
アクセントカラーにティール（#5dcaa5 / #9fe1cb）を選定: 青緑系は日本の地図記号の森林・河川を想起させ、かつ計画書にある「津久井湖・相模湖を抱く、山と水の町」のような自然描写と馴染みます。信号的な警告色を避け、旅の穏やかさを保つ配色です。
ガラス調のフローティング情報バー: 上部の市町村名と制覇カウントは backdrop-filter: blur を効かせた半透明カードで浮かせ、地図の視認性を殺さずに常時情報を表示しています。
速度と土地名の並び: 下部カードヘッダーで「相模原市 緑区」と「62 km/h」を左右に置き、『どこ』『どれくらいで動いているか』が一瞥で分かるようにしました。
「土地のたより」という呼称: "Description" や "Info" ではなく、旅情を損なわない和の呼称を使いました。ティールの細い線＋小さな英字ラベルで和欧混植のリズムを作っています。
免責と GPS 状態を最下部に分離: 「情報は目安です」は計画書の必須要件ですが、前景を邪魔しない極小フォント・セカンダリ色で配置しました。右側の「GPS 受信中」は小さなドットのパルスで生きている感を出せる位置取りです。
